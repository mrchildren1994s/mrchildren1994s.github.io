1. 更多動畫示範

Canvas 可以用來製作各種動畫，例如火柴人走路、彈跳球、旋轉方塊、粒子效果與自然模擬（如雨滴動畫）。動畫的本質是：每一幀重新繪製新的畫面，再用 requestAnimationFrame 持續更新。

2. 互動功能

Canvas 支援滑鼠事件，你可以透過滑鼠位置 (mouseX, mouseY) 讓物件跟著移動、點擊觸發動畫、或者像畫圖工具一樣拖曳繪線。這讓 Canvas 可以製作互動遊戲與工具。

3. 按鈕控制動畫

可以結合 HTML 按鈕（如開始、暫停、重設）控制動畫流程。做法是加入按鈕元素，再用 JavaScript 綁定 onclick 事件，控制動畫是否繼續更新。

4. 詳細 Canvas API 說明

Canvas 提供 2D API，包括：

  • fillRect(), strokeRect()：繪製方形
  • arc()：繪製圓形
  • beginPath(), moveTo(), lineTo()：繪線
  • drawImage()：繪圖片
  • fillText()：繪文字
  • clearRect()：清除畫布
  • requestAnimationFrame()：動畫更新

透過這些 API，便能製作圖形、文字、圖片與動畫，並進階整合為遊戲或工具。
